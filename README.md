# docker-php

[![Build Status](https://travis-ci.org/kporras07/docker-php.svg?branch=master)](https://travis-ci.org/kporras07/docker-php)

PHP Image containing some useful extensions.

## Extensions included:

- bcmath
- json
- mbstring
- simplexml
- dom
- zip
- iconv
- pdo
- gd
- xdebug
- mhsendmail
- blackfire
- intl
